# Netflix Movies SQL Project 1

## About
This SQL project analyzes Netflix movies and TV shows data using MySQL. It includes database creation, data importing, filtering, grouping, joins, and analytical queries to generate insights about genres, ratings, countries, and release trends while improving SQL and data analysis skills.
